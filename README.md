# walmart-sales-analysis-sql

## Project Overview

This project performs an in-depth **Sales Performance Analysis of Walmart Stores** using advanced MySQL queries. By leveraging a comprehensive transactional dataset, the analysis uncovers actionable business insights related to sales trends, customer behavior, operational efficiencies, and anomalies in Walmart's store operations.

## Objectives

- Identify the top branch by month-over-month sales growth rate.
- Determine the most profitable product line for each branch.
- Segment customers into spending tiers: High, Medium, Low.
- Detect anomalies in sales transactions.
- Find the most popular payment method by city.
- Analyze monthly sales distribution by gender.
- Identify preferred product lines by customer type (Member vs. Normal).
- Find repeat customers making purchases within 30 days.
- Identify the top 5 customers by sales volume.
- Analyze sales trends by day of the week.
